display:flex;
применяется для контейнера-родителя, все дивы внутри встают в ряд.
можно применить в контейнере функцию
justify-content: space-between - блоки внутри родительского встанут на расстоянии друг от друга
space-around - появится расстояние (типа margin left-right)
flex-end (start) -блоки соберутся справа ( слева) ( они же в строке)
center - блоки встанут по центру

align-items: center -блоки встанут по центру контейнера (верх-низ)
baseline - блоки поравняются на некую линию в центре блока-родителя
flex-start (end) -блоки встанут сверху(снизу) контейнера - родителя

float:left ;
применяется непосредственно для блока, который нужно поставить в строку.
при этом не забыть применить 
clear:both; желательно с помощью псевдоэлемента 
clearfix:after{
content:"";
display:table;
clear:both;
}

___________________________________

display:inline-block;
Применяется непосредственно для блока, который нужно поставить в строку.
при этом элементы выравниваются по base-line.
Выравнивать элементы по центру необходимо в блоке-родителе. text-align:center;
vertical-align
применяется для самих элементов.
но когда делаешь ссылку блоком плохо работает vertical-align(почти не работает)
